Program Python dapat dijalankan dengan beberapa mode. Jika kita mengeksekusi Python interpreter tanpa 
argumen script Python yang telah kita buat, Python interpreter akan masuk ke mode interaktif (REPL, 
read-eval-print loop).

Dengan memanfaatkan mode interaktif python kita bisa menggunakannya sebagai kalkulator.

untuk masuk kedalam mode interaktif, pada cmd cukup ketikan python.
untuk keluar dari mode interaktif, pada cmd cukup tekan CTRL + C pada keyboard.

mode interaktif biasanya ditandai dengan tanda triple right arrow >>>

contoh dari mode interaktif python

9
